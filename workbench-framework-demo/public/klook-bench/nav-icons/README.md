# Nav Icons

Place sidebar navigation SVG icons here. Name files to match the `icon` field in `src/config/nav.config.json` (e.g. `dashboard.svg`, `account-acquisition.svg`).

- **Filename** = `{icon}.svg` (no path, no extension in config)
- **Size** recommended: 32×32 px

If you previously used `public/icons/`, copy or symlink those files here for the new config-driven nav.
